그러나! 변경 안된 컴포넌트를 굳이굳이 다시 렌더링할 필요가 없으므로 react memo를 통해 설정해주자 <단 props가 변경되지 않아야함> 근데 딱히 기억안해도 된다고함. 

### *React.memo(컴포넌트)

그러면 memo로 감싼 컴포넌트의 결과를 메모하고, 만약 props가 같으면 메모한걸 그대로 가져온다. (즉 렌더링 하지 않음)

const MemorizedBtn  = React.memo(Btn)

여기서 MemorizedBtn은 props가 변경되지 않은 결과를 담고 있음. 

이때 <MemorizedBtn text = {state.change}/>처럼 변하게 되면, 그때 재 렌더링을 하게 된다. 

그리고 props가 어떤 타입으로 들어왔는지 체크해주는 패키지가 있음. 바로 PropTypes! 설치하면 됨. 

컴포넌트이름.PropTypes= {

text:PropTypes.string, fontSize: PropTypes.number}

만약 꼭 있어야 하는 값을 체크해야하면, 

PropTypes.number.isRequired 라고 추가하면됨.  선언은 렌더해주는 컴포넌트에서 해주면 됨.
